# Platform L

React + Next.js (Pages Router) + Yarn 기반 웹 프로젝트입니다.

## 시작하기

### 의존성 설치

```bash
yarn install
```

### 개발 서버 실행

```bash
yarn dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 으로 접속하세요.

### 빌드 및 프로덕션 실행

```bash
yarn build
yarn start
```

### 린트

```bash
yarn lint
```

## 프로젝트 구조

- `pages/` - 페이지 라우터 (파일 기반 라우팅)
- `styles/` - 전역 스타일
- `public/` - 정적 파일 (이미지, favicon 등)
